    3층 · 세부 원칙

    매일 지키는 구체적 약속. AI가 이 원칙으로 조언해요.

      원칙 추천 설정 · 프롬프트 직접 수정

      추천 AI의 '성격·지시문'을 고칠 수 있어요. (윤하 나이와 '정해진 출력 틀'은 앱이 자동으로 붙여줘서 고장 나지 않아요.)

      Gemini 연결 (처음 한 번만)

      발급받은 무료 API 키를 넣으면 어디서 열어도 상담이 작동해요. 키는 이 파일이 아니라 저장소에만 보관돼서 안전해요.

      AI 육아 선생님과 대화

      원칙과 윤하 나이를 바탕으로 대화하듯 이어서 상담해요. 대화는 "저장하고 새 상담"을 누르면 보관함에 통째로 저장돼요.

      상담 설정 · AI 프롬프트 직접 수정

      AI 선생님의 '성격·지시문'을 직접 고칠 수 있어요. 여기 적은 내용대로 선생님이 대답해요. (육아 원칙·윤하 나이·그동안 기록은 앱이 자동으로 함께 넣어줘요.)

      요약 저장 설정 · 프롬프트 직접 수정

      대화를 저장할 때 '어떻게 요약할지' 지시문을 고칠 수 있어요. (출력 형식과 오타 보정은 앱이 자동으로 붙여줘서 고장 나지 않아요.)

      오늘의 육아일기

      날짜는 자동으로 기록돼요. 편한 방식을 골라 남기세요.

      🎤 를 누르고 말하면 글로 적혀요. 갤럭시 키보드의 마이크 버튼을 써도 돼요.

      인스타 콘텐츠 아이디어

      그동안 쌓인 일기와 상담을 바탕으로, 인스타그램에 올릴 콘텐츠 아이디어를 뽑아줘요.

      실제 기록을 바탕으로 뽑되, 윤하의 실명·장소 같은 개인정보는 넣지 않도록 부탁해 뒀어요.

      콘텐츠 설정 · 프롬프트 직접 수정

      콘텐츠 AI의 '성격·지시문'을 고칠 수 있어요. (윤하 나이·원칙·기록은 앱이 자동으로 붙여주고, 아기 개인정보 보호 안내도 항상 함께 들어가요.)

      기록 보관함

      저장된 일기와 상담이 최근 순으로 쌓여요. 윤하의 성장을 돌아보세요.

      원칙·일기·상담 전체를 한 파일로 저장해요. 다른 기기로 옮기거나 안전하게 보관할 수 있어요. (Gemini 키는 담기지 않아요.)
